# Fast install

1. Drag the `antigravity_stage2` folder into your Antigravity project root beside `antigravity_stage1`.
2. Open a new terminal in Antigravity.
3. Run:
   `pip install -r .\antigravity_stage2\requirements.txt`
4. Run:
   `python .\antigravity_stage2\tests\test_stage2_smoke.py`
5. Confirm you see `Stage 2 smoke test passed.`
