# Stage 3 Setup

1. Copy the `antigravity_stage3` folder into the same root that already contains `antigravity_stage1` and `antigravity_stage2`.
2. Open a new terminal in Antigravity.
3. Run `pip install -r .\\antigravity_stage3\\requirements.txt`
4. Run `python .\\antigravity_stage3\\tests\\test_stage3_smoke.py`
5. Confirm the terminal prints `Stage 3 smoke test passed.`
